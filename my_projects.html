Одностраничник Пример / Single Page Application Example

Смотреть Одностраничное приложение на Vue/ See single-page application on Vue

Используется фреймвок Vue. Это простой и удобный фреймвок, обладающий огромными возможностями и легко масштабируемый до корпоративноо уровня.

Здесь я сделал простой одностраничник, приложение каталога ссылок. База данных сохраняется в обычном javascript файле, который загружается тут же в документе. Кнопка "Сохранить" позволяет сохранить всю базу в локальнео хранилище браузера. После чего она будет доступна при заходе на эту страницу только этим браузером. Немного глуповато. Но в браузере можно включить синхронизаицю для локальных хранилищ и заходить с других устройств. Кнопка "Сохранить в файл" сохраняет всю базу данных в текстовый файл в формате JSON. Потом можно будет скачать это приложение себе на компьютер и поменять файл базы данных в папке database. Лично мне было достаточно этого и не хотелось делать что-то более мощное, вроде многопользовательского серверного приложения.
